# JogoDaVelha 
Clássico jogo da velha.

## Funcionalidades
- Apresenta mensagem pop-up informando o vencedor ou empate
- Contador de Vitórias dos dois Jogadores
- Botão "Resetar" para reiniciar o jogo
- Efeitos de vitória e empate
